# Настройка Telegram бота для получения заявок

## Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и создайте бота
4. Скопируйте **токен бота** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Получение Chat ID

Есть два способа:

### Способ 1: Личные сообщения
1. Найдите бота **@userinfobot** в Telegram
2. Отправьте ему любое сообщение
3. Он ответит с вашим **Chat ID** (выглядит как `123456789`)

### Способ 2: Групповой чат
1. Создайте группу в Telegram
2. Добавьте вашего бота в группу
3. Отправьте сообщение в группу
4. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
5. Найдите `"chat":{"id":-123456789}` - это ID группы (отрицательное число)

## Шаг 3: Настройка в проекте

Откройте файл `config/settings_dev.py` и добавьте:

```python
TELEGRAM_BOT_TOKEN = 'ваш_токен_бота'
TELEGRAM_CHAT_ID = 'ваш_chat_id'
```

Пример:
```python
TELEGRAM_BOT_TOKEN = '123456789:ABCdefGHIjklMNOpqrsTUVwxyz'
TELEGRAM_CHAT_ID = '123456789'
```

## Шаг 4: Установка библиотеки requests

Если библиотека не установлена, выполните:

```bash
pip install requests
```

## Готово!

Теперь все заявки с форм будут автоматически отправляться в ваш Telegram!
